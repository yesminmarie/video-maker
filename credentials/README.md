# Credentials format

## Algorithmia

File: `algorithmia.json`

```
{
    "apiKey": "API KEY"
}

```
## Watson Natural Language Understanding

File: `watson-nlu.json`

```
{
  "apikey": "API KEY",
  "iam_apikey_description": "",
  "iam_apikey_name": "",
  "iam_role_crn": "",
  "iam_serviceid_crn": "",
  "url": "https://gateway.watsonplatform.net/natural-language-understanding/api"
}

```
